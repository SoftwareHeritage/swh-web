Advanced search
You can also refer to a directory either:
  • as a request for the content of a revision identified in absolute terms by its git SHA1.
  • as a request for the content of a revision matching a given time and place, with place the origin of the directory (and optionally its branch name), and time a timestamp of when we collected it (or as close to the timestamp as we can provide).
In all cases, you may also specify a path within the top-level directory to which you would like the results to be constrained.
